Configuration Clé d'API Gemini

Pour utiliser les interrogatoires de suspects par Intelligence Artificielle, vous pouvez configurer votre propre clé API Google Gemini. Elle sera enregistrée localement dans votre navigateur sans jamais être transmise à un serveur tiers.

Astuce : Si vous n'avez pas de clé API, le système basculera automatiquement sur un mode de dialogue d'archives pré-enregistré pour simuler l'interrogatoire sans bloquer votre jeu !
